# PETAL NYU website

Website for the org.

## Development

Install [install Ruby](https://www.ruby-lang.org/en/documentation/installation/) and run the following:
```
git clone https://github.com/petalnyu/petal-nyu.git
cd petal-nyu
gem install bundler jekyll jekyll-feed
bundle exec jekyll serve
```
